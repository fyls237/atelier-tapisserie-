# atelier-tapisserie-
Le projet concerne un atelier de menuiserie/tapisserie basé à Yaoundé (Cameroun), avec une gestion à distance depuis l’Europe. Les contraintes majeures sont : - Faible volume initial (5 à 10 commandes/mois) - Équipe réduite (sœur, cousin, futur employé) - Connexion internet instable - Budget limité
